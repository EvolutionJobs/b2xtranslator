FONT: Font Description (231h)

The workbook font table contains at least five FONT records. FONT records are numbered as follows: ifnt=00h (the first FONT record in the table), ifnt=01h , ifnt=02h , ifnt=03h , ifnt=05h (minimum table), and then ifnt=06h , ifnt=07h , and so on.  Note : ifnt=04h never appears in a BIFF file.  This is for backward-compatibility with previous versions of Excel.  If you read FONT records, remember to index the table correctly, skipping ifnt=04h .
