DCONNAME: Data Consolidation Named References (52h)

The DCONNAME record contains the complete description of a named range of cells for the Consolidate command (Data menu). The stFile field contains an encoded workbook name. For more information about this field, see “EXTERNSHEET EXB0068 ”.
