SXTH: Pivot Table OLAP Hierarchy (80Dh)

Introduced in Excel 9 (2000) this is a FRT record. It contains OLAP Pivot Table hierarchy information.

The rgb field contains a structured list of elements hidden in the PivotTable. First is a count of the number of sets. The sets follow. Each set contains a count of the number of strings in that set, followed by the strings in that set. Note : this can easily make the record larger than the limit for a single BIFF record, so one or more CONTINUEFRT records may be needed.
