WRITEACCESS: Write Access User Name (5Ch)

This record contains the user name, which is the name entered when installing Excel.

Record Data — BIFF8

Offset

Name

Size

Contents

4

stName

112

User name as an unformatted unicode string . The name is always padded with spaces so the size of the stName field is exactly 112 bytes.
