REALTIMEDATA: Real-Time Data (RTD) (813h)

Introduced in Excel 10 (2002) this BIFF record is an FRT record. There is one REALTIMEDATA record for each RTD (real-time data) topic in the workbook.  Each REALTIMEDATA record contains topic name, variant RTD data ( RTDOPER ), and an array of RTDE structures describing the set of cells associated with it.  If necessary, this record may continue with one or more CONTINUEFRT records following it.
