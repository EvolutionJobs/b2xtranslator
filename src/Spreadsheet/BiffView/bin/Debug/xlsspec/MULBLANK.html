MULBLANK: Multiple Blank Cells (BEh)

The MULBLANK record stores up to the equivalent of 256 BLANK records; the MULBLANK record is a file size optimization.  The number of ixfe fields can be determined from the ColLast field and is equal to ( colLast-colFirst+1 ). The maximum length of the MULBLANK record is (256x2+10)=522 bytes, because Excel can have at most 256 columns.  Note : storing 256 blank cells in the MULBLANK record takes 522 bytes as compared with 2,560 bytes for 256 BLANK records.
