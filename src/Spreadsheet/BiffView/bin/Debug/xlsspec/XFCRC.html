XFCRC: XF Extensions Checksum (87Ch)

This record is used in determining whether XFEXT (87Dh) records should be applied when loading a document. This record if present should come after all XF records. It contains the number of XF records that were written and a checksum of the data in those records. If the number of records and the checksum match and the document was last saved with version 12 or later then XFEXT records will be applied.
