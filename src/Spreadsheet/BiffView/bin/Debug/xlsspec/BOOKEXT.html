BOOKEXT: Extra Book Info (863h)

Introduced in Excel 10 (2002) this record is an FRT record. This record contains workbook-specific information new to Excel 10 (2002) and Excel 11 (2003). The rgf and rgf2 fields contain several bit flags and other fields, as described in the tables below.

The grbit field contains the following flags and fields:

fHidePivotTableFList

=1 if pivot table field list should be hidden for this workbook;
=0 if it should be shown.

=1 if personal information should be removed from File / Properties on save;
=0 otherwise.

3

fEmbedSmartTags

=1 if smart tags should be embedded in this workbook on save;

mdSmartTagsDisplay

Corresponding to a control in the Tools/AutoCorrect Options…./Smart Tags dialog tab:
=0 mdSmartTagsDisplayAll , if both button and indicator should be shown for each smart tag;
=1 mdSmartTagsDisplayNone , if  smart tags should not be shown at all;
=2 mdSmartTagsDisplayNoIndicator ,
if only the button (but no indicator) should be shown for each smart tag.
